How to Cite AgniCycle

If you used AgniCycle's tools to generate results in a thesis, paper, or report, please cite the specific tool(s) you used. It helps others reproduce your setup — and helps us keep building tools researchers actually rely on.

A note on versioning: AgniCycle's tools run live in the browser and update continuously rather than shipping as dated releases, so there's no archival DOI to point to. Cite the tool name, version (where one exists), and an access date — the standard approach for citing web-based software. Replace YYYY-MM-DD below with the date you actually used the tool.
AgniCycle Engine Simulator v1.3
APA (7th ed.)
Kumar, D. (2026). AgniCycle Engine Simulator (Version 1.3) [Computer software]. AgniCycle. https://agnicycle.com/simulator/
BibTeX
@misc{agnicycle_engine_simulator, author = {Kumar, Dhananjay}, title = {{AgniCycle Engine Simulator}}, year = {2026}, version = {1.3}, publisher = {AgniCycle}, howpublished = {\url{https://agnicycle.com/simulator/}}, note = {Accessed: YYYY-MM-DD} }
AgniCycle Spray Calculator Beta
APA (7th ed.)
Kumar, D. (2026). AgniCycle Spray Calculator (Beta) [Computer software]. AgniCycle. https://agnicycle.com/spray/
BibTeX
@misc{agnicycle_spray_calculator, author = {Kumar, Dhananjay}, title = {{AgniCycle Spray Calculator}}, year = {2026}, version = {Beta}, publisher = {AgniCycle}, howpublished = {\url{https://agnicycle.com/spray/}}, note = {Accessed: YYYY-MM-DD} }
AgniCycle EV Battery Suite Beta
APA (7th ed.)
Kumar, D. (2026). AgniCycle EV Battery Suite (Beta) [Computer software]. AgniCycle. https://agnicycle.com/ev-battery/
BibTeX
@misc{agnicycle_ev_battery_suite, author = {Kumar, Dhananjay}, title = {{AgniCycle EV Battery Suite}}, year = {2026}, version = {Beta}, publisher = {AgniCycle}, howpublished = {\url{https://agnicycle.com/ev-battery/}}, note = {Accessed: YYYY-MM-DD} }
Citing the platform generally

Used more than one AgniCycle tool, or referring to the platform as a whole rather than a specific result? Use this instead of (not in addition to) the tool-specific citations above.

APA (7th ed.)
Kumar, D. (2026). AgniCycle: A browser-native combustion and EV research platform [Computer software]. https://agnicycle.com
BibTeX
@misc{agnicycle_platform, author = {Kumar, Dhananjay}, title = {{AgniCycle}}, year = {2026}, publisher = {AgniCycle}, howpublished = {\url{https://agnicycle.com}}, note = {Accessed: YYYY-MM-DD} }
